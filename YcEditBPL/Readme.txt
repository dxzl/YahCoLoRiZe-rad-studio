I've pre-built the "YcEdit" bpl library which you can install in C++ Builder with:

Component->Install Packages->Add then navigate to YcEdit.bpl and install.

Then you can open YahCoLoRiZe-rad-studio project and the YcEdit component will load ok.

--------------------------

Alternatively, you can build YcEdit yourself:

Follow the Readme.md for github repository: YcEdit-rad-studio

The BPL library file will be in:
C:\Users\Public\Documents\Embarcadero\Studio\21.0\BPL

--------------------------
